# DEDKENI DEMO



### HOW TO PLAY
Keyboard:
  * Move = Arrow keys
  * Jump = Space
  * Fire = E
  * Grab/push = F

Gamepad:
  * Move = Left stick
  * Jump = Button A
  * Fire = Button X
  * Grab/push = Button Y
